# Iris Flower Classification using KNN

A beginner-level machine learning project that builds a classification model to identify iris flower species (Setosa, Versicolor, Virginica) based on 4 measurements: sepal length, sepal width, petal length, and petal width.

## Steps Covered
- Loading and exploring the Iris dataset
- Feature scaling using StandardScaler
- Splitting data into training and testing sets
- Training a K-Nearest Neighbors (KNN) classifier
- Evaluating results with a confusion matrix and F1 score

## Tech Used
Python, pandas, scikit-learn
